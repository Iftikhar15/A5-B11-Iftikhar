Question-1: What are the different ways to select an element in the DOM?

We can select elements in the DOM using getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll. Each method has its specific use cases and helps you target elements by their ID, class, tag name, or CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML includes HTML and text within an element. innerText returns only visible text, considering CSS styles. textContent includes all text content, ignoring styles and hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events for its child elements. This method leverages event bubbling to efficiently handle events on dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event starts from the target element and then propagates up to the parent elements. This allows event handlers on parent elements to react to events fired on their children.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, use document.createElement. Add it using parentElement.appendChild or parentElement.insertBefore. Remove an element with parentElement.removeChild or element.remove().
